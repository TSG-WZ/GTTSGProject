# 锐拓-地铁图书管理系统
## 项目理念
- 推崇纸质阅读理念
- 方便共享图书信息
- 快捷购买
- 提前预约
- 可借可换
## 技术栈
- ASP.NET MVC + DAPPER
- SQL SERVER 2012
- LayUI
- sqlserver
- git 管理项目

## 系统操作说明

**一、登陆：**

- PC端（地铁图书机器）：
  1. 传统登陆方式：输入账号密码点击登陆
  2. 扫码登陆方式（后期加入）：扫描微信二维码登陆

- 手机端：
  1. 微信公众号登陆--> 查找图书借阅信息，预约图书，展示附近图书馆站点信息

**二、借阅图书：**

1. 点击图书进入到图书详情页
2. 点击购买或者借阅

**三、归还图书：**

1. 登陆系统
2. 点击借阅列表--> 进行还书操作
3. 后期改进--> 于物联网结合，直接将书籍放入机器进行扫描，确认个人信息，自动归还
